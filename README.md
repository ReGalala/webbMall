Webmall is a full-stack web application that enables creators and small sellers to quickly set up their own online store without building a website from scratch.

The platform allows sellers to manage products, inventory and orders, while buyers can browse stores, search products, manage carts and place orders. Webmall handles the technical infrastructure, authentication, and data management.

The project was developed as part of the university course *DIT342 Web and Mobile Development (HT23)*.

## Tech Stack

- **Frontend:** Vue.js (Vite)
- **Backend:** Node.js / Express
- **Database:** MongoDB
- **Authentication:** Role-based (Buyer, Seller, Admin)
- **Deployment:** Node environment with Procfile support

---

## System Overview

Webmall supports three types of users:

- **Buyer**
- **Seller**
- **Admin**

### Core Features

- User authentication (buyer, seller, admin)
- Sellers can create and manage stores
- Add, edit and delete products
- Product browsing and filtering
- Shopping cart with quantity management
- Order creation and order history
- Seller/Admin order management
- Inventory management
- Role-based access control
